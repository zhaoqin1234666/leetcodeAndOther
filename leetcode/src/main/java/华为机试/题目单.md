###1 容器大小排序
    
    任给一个数组，元素有20M，1T，300G之类的，其中1T=1000G，1G=1000M
      按从小到大输出结果
      例如：输入：3
      20M
      1T
      300G
      输出：
      20M
      300G
      1T

###2 找出单独元素

    任给一个数组，其中只有一个元素是单独出现，其他是成对出现，输出单独的元素。
    例如： 输入： {2，2，1，1，4，4，7}
    输出：7
